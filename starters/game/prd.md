# 🎮 게임 기획서 (PRD) — 나만의 웹게임

> 이 파일은 **게임용 PRD 템플릿**이에요. 괄호 안 설명을 지우고 내 게임 내용으로 채우세요.
> 다 채운 뒤 Copilot에게 "이 prd.md를 보고 index.html 게임을 바꿔줘"라고 부탁하면 됩니다.

## ① 한 줄 소개
(예: 떨어지는 별을 받아 점수를 올리는 웹게임)

## ② 누가 즐길까
(예: 쉬는 시간에 심심한 친구들)

## ③ 장르 · 핵심 재미
(예: 피하기/받기 게임 · 아슬아슬하게 받는 손맛)

## ④ 조작 방법
(예: ← → 방향키 또는 화면 터치로 좌우 이동)

## ⑤ 규칙 · 승패
(예: 별을 받으면 +1점 / 폭탄에 닿으면 게임오버 / 30초 동안 최고 점수 도전)

## ⑥ 화면에 보이는 것
(예: 점수, 남은 시간, 플레이어, 떨어지는 별과 폭탄, 게임오버 화면)

## ⑦ 난이도 변화
(예: 시간이 지날수록 별이 더 빨리 떨어짐)

## ⑧ 분위기 · 그래픽 · 사운드
(예: 밤하늘 배경, 반짝이는 별, 받을 때 '딩동' 효과음)

---

## 🚀 이 스타터로 시작하는 법
1. `index.html`을 브라우저로 열면 **별 받기 게임**이 바로 실행돼요.
2. 위 PRD를 내 게임으로 채워요.
3. Copilot Chat에 아래처럼 하나씩 부탁하며 키워요.

```
지금 이 index.html은 별 받기 게임이야.
prd.md를 참고해서 아래 기능을 하나 추가해줘:
- (예: 빨간 폭탄을 추가하고, 닿으면 게임오버 화면 보여주기)
초보용 한국어 주석을 달고, 바뀐 전체 코드를 보여줘.
```

## 💡 확장 아이디어
- 목숨(하트) 3개, 다 잃으면 게임오버
- 남은 시간 타이머와 최고 점수 저장
- 별 종류마다 다른 점수(금별 +5)
- 배경음악과 효과음, 파티클(반짝이) 효과
