## 📦 이미 든 아이콘 (`icons/` 폴더 · 바로 사용)
`<img src="assets/icons/heart.svg" width="24">` 처럼 쓰거나, 파일을 열어 SVG 코드를 복사해 붙여요. 색은 `currentColor`라 CSS `color`로 바뀌어요.
