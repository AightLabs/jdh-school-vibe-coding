## 🚀 이 스타터로 시작하는 법
1. `index.html`을 브라우저로 열면 **소개 페이지**가 바로 보여요.
2. 위 PRD를 내 내용으로 채워요.
3. OpenCode에 아래처럼 하나씩 부탁하며 꾸며요.

```
지금 이 index.html은 Tailwind로 만든 소개 페이지야.
prd.md를 참고해서 아래를 바꿔줘:
- (예: 제목을 내 이름으로, 좋아하는 것 3가지를 실제 내용으로,
       색을 초록 계열로 바꾸기)
초보용 한국어 주석을 달고, 바뀐 전체 코드를 보여줘.
```
